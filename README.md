# 许兵的个人博客

这是许兵的个人博客，包含以下板块：算法、股票数据、杂谈和其他。

## 目录结构

blog/
├── index.html
├── algorithm.html
├── stocks.html
├── misc.html
├── others.html
├── styles.css
└── scripts.js


## 文件说明

- `index.html`: 博客主页
- `algorithm.html`: 算法板块页面
- `stocks.html`: 股票数据板块页面
- `misc.html`: 杂谈板块页面
- `others.html`: 其他板块页面
- `styles.css`: 博客的样式表
- `scripts.js`: 博客的JavaScript文件
